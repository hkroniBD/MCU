# üî∑ Common Features of a Microcontroller

These are the building blocks that exist in almost all modern microcontrollers, whether it is Arduino (ATmega), PIC, STM32, or ESP32.

## üß© 1Ô∏è‚É£ Central Processing Unit (CPU)

**Definition**
The **CPU** is the brain of the microcontroller. It fetches instructions, processes data, and controls all operations.

**Functions**

* Executes program instructions
* Performs arithmetic and logic operations
* Controls data flow between peripherals

**Physical meaning:**
This is the part that decides **when to turn ON a motor**, **when to read a sensor**, or **when to send data**.

---

## üß© 2Ô∏è‚É£ Program Memory (Flash / ROM)

**Definition**
This memory stores the **firmware or program** that you write.

Example:
In Arduino, this memory stores your `.ino` sketch.

**Physical meaning:**
This is like the **brain memory** of the device ‚Äî without it, the MCU does nothing.

---

## üß© 3Ô∏è‚É£ Data Memory (RAM)

**Definition**
RAM stores **temporary data**, variables, sensor readings, and calculations.

**Physical meaning:**
When you measure temperature or voltage, that value is stored here before being used or sent.

---

## üß© 4Ô∏è‚É£ Input / Output (I/O) Pins

**Definition**
GPIO (General Purpose Input Output) pins allow the MCU to interact with the outside world.

| Mode   | What it does                    |
| ------ | ------------------------------- |
| Input  | Reads switch, sensor, signal    |
| Output | Drives LED, motor relay, buzzer |

**Physical meaning:**
These pins are the **hands and ears** of the microcontroller.

---

## üß© 5Ô∏è‚É£ Timers and Counters ‚è±Ô∏è

**Definition**
Timers generate delays, measure time, count pulses, and generate PWM signals.

**Typical uses**

* Motor speed control
* Digital clocks
* Pulse width modulation
* Event counting

**Example:**
If a motor needs 1000 pulses per second, timer hardware ensures it precisely.

---

## üß© 6Ô∏è‚É£ ADC (Analog to Digital Converter)

**Definition**
ADC converts analog voltages into digital numbers.

Equation (visual format):

Digital value = (Vin / Vref) √ó (2^n ‚àí 1)

Where
Vin = input voltage
Vref = reference voltage
n = ADC resolution (8, 10, 12 bit etc.)

**Physical meaning:**
Allows the MCU to read **temperature**, **light**, **pressure**, **battery voltage** etc.

---

## üß© 7Ô∏è‚É£ Communication Interfaces

Microcontrollers can talk to other devices using:

| Interface | Used for              |
| --------- | --------------------- |
| UART      | PC, GPS, Bluetooth    |
| I2C       | Sensors, LCD          |
| SPI       | SD card, ADC, display |
| CAN       | Automotive systems    |
| USB       | Data and power        |

**Physical meaning:**
These are the **language translators** of the MCU.

---

## üß© 8Ô∏è‚É£ Clock Oscillator ‚è≥

**Definition**
Provides timing signal for instruction execution.

Clock frequency determines speed:

Instruction time = 1 / Clock frequency

Example:
16 MHz ‚Üí 62.5 ns per cycle

**Physical meaning:**
This is the **heartbeat** of the microcontroller.

---

## üß© 9Ô∏è‚É£ Interrupt System

**Definition**
Interrupts allow the MCU to respond instantly to external or internal events.

Example:
Button pressed ‚Üí interrupt ‚Üí MCU stops current work ‚Üí handles button.

**Physical meaning:**
This gives the MCU **reflexes** like a human.

---

## üß© üîü Watchdog Timer üõ°Ô∏è

**Definition**
A safety timer that resets the MCU if software gets stuck.

**Physical meaning:**
Acts like a **self-healing circuit** to avoid system freeze.

# üè≠ Real-World Example

In a **smart irrigation system** üå±

* ADC ‚Üí reads soil moisture
